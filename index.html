Anotações de Estudo: Formatações Adicionais em HTML

1. Introdução

Além dos títulos, parágrafos, negrito e itálico, o HTML oferece uma variedade de tags para formatar o texto de formas mais específicas. Elas ajudam a dar mais significado ao conteúdo e melhorar a apresentação da informação.

2. Exemplos Práticos e Explicações

<small> - Texto Pequeno

Usado para indicar texto secundário ou menos importante.

Texto principal. Copyright 2025. Todos os direitos reservados.

<del> - Texto Deletado

Indica que o texto foi removido. Útil para mostrar edições.

Preço antigo: R$ 100,00 Preço novo: R$ 80,00

<ins> - Texto Inserido

Indica que o texto foi adicionado em uma revisão.

Preço antigo: R$ 100,00 Preço novo: R$ 80,00

<mark> - Texto Marcado

Destaca partes importantes do conteúdo.

Veja o capítulo 3 para mais detalhes.

Sua busca por "banana" encontrou banana nanica e banana prata.

<sub> - Subscrito

Usado para fórmulas químicas ou matemáticas.

A fórmula da água é H2O.

<sup> - Sobrescrito

Usado para expoentes ou referências.

A energia é E = mc2.

Hoje é dia 20o do mês.

<abbr> - Abreviação

Melhora a acessibilidade ao indicar o significado de abreviações.

Estamos estudando HTML.

<bdo> - Direção de Texto

Altera a direção do texto exibido.

Exemplo: texto invertido

<pre> - Texto Pré-formatado

Preserva espaços, quebras de linha e tabulação. Ideal para código.

function greet(name) {
    console.log("Hello, " + name + "!");
}
greet("World");

3. Conclusão

Essas tags enriquecem o HTML com semântica e estrutura, facilitando a leitura, manutenção e acessibilidade do conteúdo. Use-as com sabedoria!
